## Instructions

Follow these steps whenever frontend development or UI building is required.

### 1. Project Setup
- Initialize frontend framework (Next.js, React, etc.)
- Configure TypeScript if needed
- Set up styling system (CSS, Tailwind, tsParticles, Framer Motion, Lucide icons, CSS Modules)
- Organize folder structure (pages, components, layouts)

### 2. Page Creation
- Create pages based on routes
- Use reusable layouts
- Fetch and display data properly
- Handle loading and error states

### 3. Component Development
- Build small, reusable components
- Keep components focused on a single responsibility
- Pass data via props
- Use state and hooks where necessary

### 4. Layout Structure
- Define global layout (header, footer, sidebar)
- Ensure responsive design
- Use grids and flexbox effectively
- Maintain consistent spacing and typography

### 5. Styling
- Apply consistent color themes
- Use responsive units
- Handle hover, focus, and active states
- Avoid inline styles when possible

### 6. State & Interaction
- Handle user interactions (clicks, forms)
- Manage local and global state
- Provide visual feedback
- Ensure accessibility

### 7. Performance & Best Practices
- Optimize images and assets
- Avoid unnecessary re-renders
- Keep UI simple and clean
- Test on multiple screen sizes
